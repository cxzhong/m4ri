### NAME

cpucycles-open - open up cycle counting

### SYNOPSIS

    cpucycles-open

### DESCRIPTION

`cpucycles-open`
tries to change some aspects of OS configuration
to open up higher-quality cycle counters for `cpucycles`.

Some system administrators may wish to configure
`cpucycles-open` to be run on each boot.
However,
what `cpucycles` does is contrary to some security policies.

`cpucycles-open`
often improves cycle counting,
but does not guarantee success.
